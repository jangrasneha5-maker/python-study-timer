# python-study-timer
Pomodoro-style timer with ASCII art for AutoCAD practice sessions.

Purpose

The primary goal of this project is to provide a distraction-free productivity interface. By leveraging the Pomodoro technique—alternating focused sprints with short breaks—the timer helps users maintain a "flow state," particularly during repetitive or complex design tasks in AutoCAD where time management is critical.


✨ Key Features

Customizable Intervals: Set your own durations for study blocks and break periods to suit your workflow.

Dynamic ASCII Art: Features visual study topics rendered in ASCII art to provide a unique, terminal-based aesthetic.

Session Logging: Automatically tracks and logs time spent per topic, allowing for post-study performance analysis.

Lightweight CLI: A simple, fast, and responsive Command-Line Interface that consumes minimal system resources.
